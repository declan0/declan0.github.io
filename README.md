# Declan O'Reilly — personal website

This is a static website built with plain HTML, CSS, and a small amount of JavaScript.
No build step or framework is required.

## Preview locally

Open a terminal in this folder and run:

```bash
python3 -m http.server 8000
```

Then visit [http://localhost:8000](http://localhost:8000).
Edits appear after refreshing the browser.

## Edit the site

- `index.html` — homepage and bio
- `research.html` — research projects
- `videos.html` — video cards and links
- `cv.html` — CV page
- `styles.css` — colors, type, spacing, and layout
- `CV.pdf` — replace this file with a new PDF using the same filename to update the CV

Search for the HTML comments in `research.html` and `videos.html` for quick editing notes.

## Publish with GitHub Pages

Push the contents of this folder to the `declan0.github.io` repository. For a user
site repository with that exact name, GitHub Pages serves the site from the default
branch automatically (or it can be enabled under **Settings → Pages**).
